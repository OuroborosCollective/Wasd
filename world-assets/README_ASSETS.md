# World Assets

**Workflow:** Dateien hier ablegen → `pnpm run dev` oder `pnpm run build` (sync läuft automatisch) → URLs **`/world-assets/<unterordner>/<datei>.glb`**. Details: **`HOW_TO_ADD_GLBS.md`**.

Lege hier GLB-Assets ab:
- characters/
- monsters/
- buildings/
- props/
- vegetation/
- admin/

Beispiele:
- uschi_gossip.glb
- warrior_male.glb
- warrior_female.glb
- castle_gate.glb
- castle_watch_tower.glb
- tree_oak_spring.glb
- gm_giraffe.glb
