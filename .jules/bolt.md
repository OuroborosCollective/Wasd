## 2026-03-21 - [Array Iteration Optimization]
**Learning:** Performing multiple iterations over the same array to count items and selectively remove them (e.g., combining `.filter().length` with multiple `.findIndex()` and `.splice()` calls) creates redundant passes and intermediate array allocations, degrading performance, especially on hot paths like player interactions.
**Action:** Replace multiple chained array methods and lookup calls with a single backwards loop to count and selectively splice items in one pass, eliminating intermediate array allocations and redundant iterations.
