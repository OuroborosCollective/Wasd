## 2025-03-19 - Add ARIA label, role="log", and aria-live to Chat Messages
**Learning:** For dynamic text containers that continuously update with new information (like chat boxes), simply adding `aria-label` isn't enough. Screen readers need `role="log"` and `aria-live="polite"` to correctly announce new incoming messages without rudely interrupting the user.
**Action:** When auditing or implementing UI components that display streaming or updating information (logs, chats, event feeds), always include `role="log"` and an appropriate `aria-live` value alongside `aria-label`.
